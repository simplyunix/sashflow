**SASHflow** is an intelligent DJ mixing engine for electronic music,
focused on house, trance, and dance anthems.

## Vision
Automated DJ mixing using:
- BPM detection
- Key detection (harmonic mixing)
- Drop prediction
- Energy profiling
- Structural segmentation
- Flow-based transitions

## Features (current)
- BPM detection
- Key detection

## Roadmap
- Drop detection
- Phrase & structure detection
- Energy mapping
- Auto-transition engine
- Mix rendering

## Project Name
SASHflow = SASH + Flow State Mixing Engine
